Twit.js

I'm not going to lie to you, this isn't really a test ‘console’ as a nearly empty HTML file that loads twit.js along with its prerequesites, so that you can test use from inside the Webkit console.

To get started, give this a go:

> twttr = new TwitJS(consumer_key, consumer_secret, { debug: true });
> twttr.getAuthorizationUrl(twttr.log);
> twttr.authAccessToken(oauth_oob/verifier, twttr.log);
> twttr.statusesHomeTimeline(null, twttr.log);

That'll instantiate the object, get you an Authorization URL for your app, which you should then visit in a new window, then return to the console and verify using the OOB/verify token you obtained after authorization.

twttr.log wraps around console.log by default; determined by passing {debug:true} in the constructor.
